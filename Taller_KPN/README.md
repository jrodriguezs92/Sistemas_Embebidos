## Descripción
Ejemplos básicos redes de procesos de Kahn codificados en Python. Factorial y secuencia de Fibonacci.

## Ambiente de prueba
Linux, Ubuntu 18.04.

## Instrucciones
Se necesita Python 3.
### Ejecución
En la carpeta raiz ejecutar:  
```
$ python3 fibonacci.py
```
o:
```
$ python3 factorial.py
```
